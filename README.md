# AIxGRC-VET Prototype

## Overview
AIxGRC-VET is an AI-powered Governance, Risk, and Compliance platform designed to determine whether an organization's workforce is ready to safely deploy high-risk AI systems before go-live.

## Prototype Features
- Workforce Readiness Index (WRI)
- Behavioral Simulation Analysis
- Deployment Risk Assessment
- AI-Generated Recommendations
- Go/No-Go Decision Support

## Technologies
- Python
- Pandas
- Plotly
- Jupyter Notebook
